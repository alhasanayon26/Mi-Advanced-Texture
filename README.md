# Advanced Textures Enabler

A lightweight systemless module that unlocks **Advanced Textures** on supported **Xiaomi, Redmi, and POCO** devices. Fully compatible with both **Magisk** and **KernelSU**, this module enables premium system textures that are normally restricted to higher-end devices, delivering a more refined and visually appealing user interface while maintaining excellent performance and stability.

Many Xiaomi devices disable advanced visual effects based on hardware classification rather than actual capability. This module modifies the necessary system configurations to unlock these premium textures, allowing compatible devices to enjoy a richer and more polished HyperOS experience without permanently modifying the system partition.

---

## Features

- ✨ Enables Advanced Textures
- 🎨 Unlocks premium system visual effects
- 📱 Supports Xiaomi, Redmi, and POCO devices
- ⚡ Lightweight and performance-friendly
- 🔒 Fully compatible with both **Magisk** and **KernelSU**
- ♻️ Systemless installation with easy removal
- 🚀 Enhances the overall HyperOS user interface
- 💯 Stable and optimized for everyday use

---

## Requirements

- HyperOS
- Root access
- **Magisk** or **KernelSU**

> **Compatibility:** This module is compatible with most Xiaomi, Redmi, and POCO devices running supported HyperOS versions. Actual functionality may vary depending on the device model and software version.

---

## Installation

### Magisk

1. Download the latest release.
2. Open the **Magisk** application.
3. Navigate to **Modules**.
4. Tap **Install from Storage**.
5. Select the downloaded ZIP file.
6. Reboot your device.

### KernelSU

1. Download the latest release.
2. Open the **KernelSU Manager** application.
3. Navigate to **Modules**.
4. Install the downloaded ZIP file.
5. Reboot your device.
